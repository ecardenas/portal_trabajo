# Frontend Público Skill

Usa este skill para construir o mejorar vistas HTML/CSS/JS del portal de convocatorias.

Objetivo:
Crear una interfaz amigable para búsqueda y consulta de convocatorias.

Reglas UI/UX:
- Diseño limpio, profesional y responsive.
- Buscador visible.
- Filtros por entidad, puesto, ubicación, modalidad, fecha y estado.
- Tarjetas o tabla de resultados.
- Indicador de cantidad de resultados.
- Estados visuales: vigente, vence pronto, vencido, nuevo.
- No depender de frameworks pesados si el proyecto usa HTML/CSS/JS vanilla.
- Consumir endpoints FastAPI mediante fetch().

Cuando generes código:
1. Entregar HTML, CSS y JS separados si corresponde.
2. Usar nombres de clases claros.
3. Evitar estilos inline.
4. Añadir manejo de errores al consumir API.
